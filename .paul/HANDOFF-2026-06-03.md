## Important Notes

- Stack is **FastAPI** (not Django — README says Django but code is FastAPI + Jinja2 + HTMX + SQLite)
- `SANDBOX_MODE=none` required on macOS — student code runs with timeout only, no namespace isolation
- Assignment must have **GitHub enabled** checkbox checked for professor submission to work
- GitHub repo must be **public** (or GITHUB_TOKEN set for private repos)
