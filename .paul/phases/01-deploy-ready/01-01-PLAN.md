<task type="auto">
  <name>Task 1: Create start.sh</name>
  <files>start.sh</files>
  <action>
    Create a shell script that:
    1. Loads `.env` if it exists (via `set -a; source .env; set +a`)
    2. Starts `python grader/worker.py` in background, captures PID
    3. Starts `uvicorn main:app --host 0.0.0.0 --port 8000` in foreground
    4. On exit (trap SIGINT SIGTERM), kills the worker PID and waits
    5. Print startup message: "AutoGrader starting — http://0.0.0.0:8000"

    Make executable (chmod +x is handled by the file permissions note — include `chmod +x start.sh` in verify step).
  </action>
  <verify>bash -n start.sh && ls -la start.sh | grep -q x && echo "OK"</verify>
  <done>AC-1 satisfied: single command launches both processes</done>
</task>
